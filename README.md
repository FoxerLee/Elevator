### 项目要求

某一栋楼共有20层，五部互相关联的电梯，基于线程的思想，模拟实现一个电梯调度的程序。



### 开发环境

- 系统：macOS Sierra (version 10.12.4)
- IDE：Qt Creator 4.2.1, Based on Qt 5.8.0 (Clang 7.0 (Apple), 64 bit)
- 语言： C++
